20 — Verifying "absence" security properties

Verification approach Unresolved
Affects: REQ-SEC-002 (login password never read), REQ-SEC-006 (transient in-memory decryption), REQ-RECORD-006 (key zeroed on exit)

The gap

These are negative guarantees — "X never happens." You can't prove a universal negative with a single CLI call; each needs its own technique, and some are only partially automatable.

(iii) Code review that no login-auth path is called. 		Audit + construction

Recommendation

Split this requirement set explicitly into automated (transient-decryption FS watch) and manual/audit (login-password path, key zeroing) acceptance items so the test plan is honest about coverage.
